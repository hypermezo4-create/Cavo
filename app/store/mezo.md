# app/store

**هنا إيه؟** صفحات المتجر التي يراها العميل.

**لو هتضيف هنا:** أضف صفحة عرض جديدة داخل المتجر مثل قسم أو صفحة محتوى.

**لو عدلت هنا:** شكل تجربة العميل ومسارات المتجر سيتغيران مباشرة.

**ملاحظة:** تم حذف cart / checkout / order-confirmation لأن الموقع الحالي يعمل كعرض وواجهة روابط، وليس شراء مباشر من الويب.
