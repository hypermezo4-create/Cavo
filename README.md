# Cavo Pro Premium Showcase

Premium showcase-only footwear site for Cavo.

## What this version does
- Showcase-only catalog with no prices on the website
- Men, Women, Kids, and Offers collections
- Premium homepage with smooth motion and subtle 3D feel
- Reviews page with demo login required for submission
- Contact links page is the only place that points visitors toward communication/app flow

## Run locally
```bash
npm install
npm run dev
```

## Deploy
Use Vercel or any standard Next.js host.
No database is required for this version.
